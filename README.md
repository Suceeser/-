# Zhihu_Spider

知乎回答评论爬取

## 依赖
- node.js 
- python
- npm install -g jsdom
- npm install -g crypto-js
- pip install requests
- pip install pyexecjs2

## 运行
pycharm打开文件fetch_comment.py 右键运行
